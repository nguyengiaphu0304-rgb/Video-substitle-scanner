# Video Subtitle Scanner v1.0.0

Video Subtitle Scanner extracts accessible timed captions from the active browser tab and exports SRT, WebVTT, CSV or notes. Processing stays in the browser; Advanced Scan is user-started and requests optional debugger and origin access only when needed.

## Highlights

- Unicode-aware parsing for SRT, WebVTT and TTML caption sources.
- Optional-permission lifecycle with deterministic browser tests for denial, restricted pages and debugger detach.
- Keyboard, focus, live-status and automated axe coverage for the popup.
- Reproducible cross-platform extension ZIP with strict allowlisting, integrity sidecars and SHA-256 verification.
- Project-owned synthetic demo evidence with source-to-output lineage.
- Large-fixture parser and package-size regression budgets.
- Automated privacy audit for permission drift, fixed endpoints and remote-code primitives.

## Privacy and permissions

The extension does not send captions to the developer or an analytics service. Advanced Scan may fetch caption resources directly from the active page's origin or CDN after the user grants optional access; that provider can observe normal browser request metadata. All executable extension code is included in the package.

## Limitations

- Accessible tracks and discoverable network captions vary by provider, authentication state and DRM restrictions; compatibility with every video service is not claimed.
- Synthetic fixtures verify deterministic behavior but are not evidence from a real provider or user video.
- OCR is not included in this release.
- The performance timing is a CI regression gate, not a universal device benchmark.
- Chrome Web Store publication is not part of this release candidate.
- Manual permission-prompt, privacy, 200% zoom/reflow and screen-reader observations must be recorded before tagging the final release.

This software is an accessibility and research utility. Review extracted captions against the source before relying on them.

# v1.0.0 release checklist

## Automated gates

- [x] Package and extension versions are `1.0.0`.
- [x] Permission and executable-source privacy policy is enforced by `npm run privacy:verify`.
- [x] Parser, export, packaging, demo, manifest and performance tests are deterministic.
- [x] Browser integration covers permission denial, restricted pages, extraction and debugger detach.
- [x] CI runs lint, type-check, tests, build, audit and artifact verification.

## Human-observed gates

- [ ] Observe and record the Chrome optional permission prompt.
- [ ] Review the packaged extension against `docs/privacy-policy.md`.
- [ ] Verify popup keyboard operation and 200% zoom/reflow in Chrome.
- [ ] Complete and record at least one supported screen-reader session.

## Publication gates

- [ ] Confirm every automated check passes on the final merge commit.
- [ ] Confirm every human-observed gate above has evidence.
- [ ] Create annotated tag `v1.0.0` at that exact merge commit.
- [ ] Publish a non-prerelease GitHub Release using `docs/releases/v1.0.0.md`.

Status: **release candidate; not released**. Do not check a human-observed item from automated evidence.
